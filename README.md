# SacrificeUI

World of Warcraft addon for **Sacrifice** guild on US-Hyjal.

A single addon that manages dependencies, checks required addons are installed, and provides contextual M+ dungeon helpers.

## Features

- **Addon Checker** — On login, shows a window listing required and suggested addons with install status. Window stays visible until all required addons are loaded.
- **M+ Dungeon Helper** — Contextual tips window when entering a Mythic+ dungeon: key interrupts, dangerous abilities, dispels, buffs, and class-specific notes.
- **Slash Commands**
  - `/sacrifice` — Toggle the main addon checker window
  - `/sacrifice m+` — Show the M+ dungeon helper for your current dungeon

## Required Addons

These must be installed for raiding/M+ with Sacrifice:

- [BigWigs Boss Mods](https://www.curseforge.com/wow/addons/big-wigs)
- [Little Wigs](https://www.curseforge.com/wow/addons/little-wigs)
- [Method Raid Tools](https://www.curseforge.com/wow/addons/method-raid-tools)
- [Clickable Raid Buffs](https://www.curseforge.com/wow/addons/clickable-raid-buffs)
- [Khesyc's iLvl](https://www.curseforge.com/wow/addons/khesycs-ilvl)

## Suggested Addons

- [RaiderIO](https://www.curseforge.com/wow/addons/raiderio)
- [Cursor Trail](https://www.curseforge.com/wow/addons/cursor-trail)
- [WarpDeplete](https://www.curseforge.com/wow/addons/warpdeplete)
- [Premade Group Filter](https://www.curseforge.com/wow/addons/premade-groups-filter)

## Installation

1. Download or clone this repo into your WoW addons folder:
   ```
   World of Warcraft/_retail_/Interface/AddOns/SacrificeUI
   ```
2. Restart WoW or `/reload`
